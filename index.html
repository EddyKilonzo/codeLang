What is programming?

Put simply, programming is giving a set of instructions to a computer to execute. If you’ve ever cooked using a recipe before, you can think of yourself as the computer and the recipe’s author as a programmer. The recipe author provides you with a set of instructions which you read and then follow. The more complex the instructions, the more complex the result!

JavaScript

Javascript is used by programmers across the world to create dynamic and interactive web content like applications and browsers.
JavaScript is so popular that it's the most used programming language in the world, used as a client-side programming language by 97.0% of all websites.

Ruby

Ruby is most used for building web applications. However, it is a general-purpose language similar to Python, so it has many other applications like data analysis, prototyping, and proof of concepts.
Probably the most obvious implementation of Ruby is Rails web, the development framework built with Ruby.

Python

Python is commonly used for developing websites and software, task automation, data analysis, and data visualization.
Since it's relatively easy to learn, Python has been adopted by many non-programmers such as accountants and scientists, for a variety of everyday tasks, like organizing finances.

C++

C++ is a powerful general-purpose programming language. It can be used to develop operating systems, browsers, games, and so on.
C++ supports different ways of programming like procedural, object-oriented, functional, and so on. This makes C++ powerful as well as flexible.
